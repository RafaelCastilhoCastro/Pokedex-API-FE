# Pokedex API Frontend

## Projeto Frontend de uma Pokedex desenvolvido em React.Js que se alimenta de uma API. :iphone:

### Como usar
- Clone o repositório
- Rode `npm i` (ou equivalente) para instalar as dependências        
- Rode `npm run build`
- Rode `npm run start`

### Dependências
- Axios
- React
- React-router-dom
- React-toastify
- Styled-components
